# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. It's built with HTML, CSS, and JavaScript, and it's designed to be a fun and easy-to-use tool for anyone who wants to try their luck at the lottery.

## Features

*   **Generate Random Numbers:** Click a button to generate a set of random lottery numbers.
*   **Modern Design:** The application features a clean and modern design with a user-friendly interface.
*   **Responsive:** The layout is responsive and works well on both desktop and mobile devices.

## Current Plan

*   **HTML Structure:** Create the basic HTML structure for the application, including a title, a button to generate numbers, and a container to display the numbers.
*   **CSS Styling:** Style the application with a modern and visually appealing design. Use CSS variables for easy theming.
*   **JavaScript Logic:** Implement the JavaScript logic to generate random lottery numbers and display them on the page.
